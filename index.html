ProofLock

ProofLock is an offline software tool that creates tamper-evident proof for screenshots and screen recordings.

ProofLock allows users to lock image or video files by generating a cryptographic hash and timestamp. Once locked, any change to the file can be detected.

ProofLock works entirely on the user's device and does not require accounts, cloud services, or data uploads. Users purchase proof credits to lock media files locally.

How It Works

  • Select an image or screen recording
  • Generate cryptographic proof and timestamp
  • Optionally apply a local digital signature
  • Verify file integrity offline at any time
